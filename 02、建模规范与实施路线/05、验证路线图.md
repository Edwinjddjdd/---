# 涡环模型验证路线图

## 1. 验证顺序

```text
接口符号
  -> 基线诱导速度
  -> Johnson 垂直下降曲线
  -> 水平速度边界
  -> 动态滞后
  -> Simulink 独立测试台
  -> 接入已有基础模型
```

## 2. MATLAB 单函数验证

| 对象 | 最低检查 |
|---|---|
| 参数文件 | 单位完整，参数来源有注释 |
| 入口换算 | $V_d$ 与 $V_z$ 符号相反，$V_H\ge0$ |
| 基线诱导速度 | 悬停、上升和正常下降输出有限 |
| Johnson 基线 | 两端平滑接回动量理论分支 |
| VRS 增量 | D、N、X、E 处函数值连续 |
| 边界斜率 | N、X 处总入流斜率为零 |
| 水平速度门控 | $V_H$ 增大时 VRS 区域收缩 |
| 动态滞后 | 阶跃响应无跳变、无发散 |

## 3. Simulink 独立场景

| 场景 | 输入 | 预期现象 |
|---|---|---|
| 悬停 | $V_H=0$，$V_d=0$ | $v_i$ 稳定在悬停量级，`vrs_flag=0` |
| 小下降率 | $V_H=0$，缓慢增加 $V_d$ | 仍处正常区，无错误触发 |
| 垂直进入 | $V_H=0$，$V_d$ 进入 Johnson 区间 | VRS 增量出现，状态标志有效 |
| 深度下降 | 继续增加 $V_d$ | 曲线通过 VRS 区并平滑接回基线 |
| 水平改出 | 保持 $V_d$，增加 $V_H$ | VRS 强度衰减并退出 |
| 动态响应 | 对 $V_d$ 或 $T$ 做阶跃 | $v_i$ 以时间常数 $\tau$ 平滑响应 |

## 4. 关键记录量

- 时间 $t$。
- $V_H$、$V_d$ 和 $V_z$。
- $T$、$\rho$、$R$ 和 $\Omega$。
- $v_h$、$\mu$ 和 $\nu$。
- 基线诱导速度 $v_{\mathrm{base}}$。
- VRS 增量 $\Delta v_{\mathrm{VRS}}$。
- 准定常目标 $v_{i,\mathrm{QS}}$。
- 动态诱导速度 $v_i$。
- 总轴向入流 $q$。
- `vrs_flag`、`vrs_strength` 和输入有效标志。

## 5. 文献对比

第一版至少完成两类对比：

1. 用 Johnson 标称参数复现无量纲平均入流曲线和稳定边界。
2. 用王畅等 2023 年风洞试验边界作为独立危险区对照。

两者用途不同：Johnson 模型负责生成平均入流和动态响应，王畅等边界负责检查危险区位置。不能把两个公式直接相加。

## 6. 接入已有模型后的检查

- 关闭 VRS 修正时，已有模型结果应与接入前基本一致。
- 开启 VRS 后，诱导速度变化只通过约定接口作用一次。
- 若已有主旋翼模型已根据 $v_i$ 计算拉力，不再额外乘一个无来源的拉力损失系数。
- 反馈路径中必须有入流积分状态，避免形成纯代数环。
- 改出测试先检查 $V_H$ 增大是否使状态离开 VRS 区，再看整机高度和姿态响应。
