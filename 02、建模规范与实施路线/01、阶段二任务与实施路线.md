# 阶段二任务与实施路线

## 1. 本阶段只做什么

本阶段只建立涡环状态子系统，不建立六自由度非线性机体模型，也不重复实现完整主旋翼、尾桨、机身和尾翼模型。

外部已有基础模型提供飞行状态和旋翼载荷；本项目输出涡环修正后的诱导速度及诊断量。

## 2. 总体数据流

```text
已有基础模型
  -> 接口换算：得到桨盘面内速度、下降率、拉力
  -> 基线诱导速度
  -> Johnson 涡环准定常修正
  -> 一阶动态入流滞后
  -> 修正诱导速度和 VRS 状态
  -> 反馈到已有旋翼/飞行动力学模型
```

## 3. 阶段划分

| 阶段 | 工作 | 交付物 |
|---|---|---|
| A | 冻结接口和符号 | 输入/输出表、单位、正方向 |
| B | MATLAB 静态算法 | 基线诱导速度、Johnson VRS 增量和准定常目标函数 |
| C | MATLAB 曲线验证 | 垂直下降曲线、连接点连续性、边界点检查 |
| D | Simulink 独立测试台 | `VRS_Inflow` 子系统与场景输入 |
| E | 接入已有模型 | 速度与拉力输入、诱导速度反馈、数据记录 |
| F | 场景和对比验证 | 进入、发展、改出和边界扫掠结果 |

## 4. 必须先冻结的接口

1. 桨盘面内速度 $V_H$ 的单位为 $mathrm{m/s}$，大小不小于 $0$。
2. 下降率 $V_d$ 规定向下为正，单位为 $mathrm{m/s}$。
3. 主旋翼拉力 $T$ 规定为正标量，单位为 $mathrm{N}$。
4. 诱导速度 $v_i$ 规定沿悬停诱导流方向为正，单位为 $mathrm{m/s}$。
5. 已有模型与本模块之间只在一个位置进行坐标和符号换算。
6. 第一版的旋翼半径 $R$、转速 $\Omega$ 和空气密度 $\rho$ 作为参数，不做成随时间变化的端口。

## 5. 推荐实施顺序

1. 只在 MATLAB 中画出正常入流曲线。
2. 复现 Johnson 垂直下降准定常入流曲线。
3. 检查 D、N、X、E 连接点的函数值和斜率。
4. 加入水平速度对 VRS 区域的收缩作用。
5. 加入一阶动态入流滞后。
6. 建立 Simulink 独立测试台。
7. 接入已有基础模型。
8. 最后再考虑低频扰动、Pitt-Peters 或更高保真模型。

## 6. 当前不做

- 六自由度非线性刚体方程与姿态积分。
- 尾桨、机身、平尾和垂尾气动力重建。
- 完整配平器重建。
- 自由尾迹、CFD 或高阶桨盘非均匀入流。
- 没有文献依据的随机拉力损失公式。
