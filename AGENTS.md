# 项目级协作提示

- 本项目所有 Markdown 文档（`.md`）中，凡涉及数学公式、变量定义、矩阵、微分方程、传递函数、积分、求和、上下标等数学表达式，统一写成 LaTeX 格式。
- 行内公式使用 `$...$`，例如 `$V_i$`、`$\dot{x}=Ax+Bu$`。
- 独立成行或多行公式使用 `$$...$$`，例如：

$$
\dot{x}=Ax+Bu
$$

- 不使用 Word 公式截图、纯文本近似写法或无法渲染的非 LaTeX 公式表达。

## 项目启动流程

- 当用户说“启动项目”“开始项目”“进入项目模式”“帮我把这个项目跑起来”“开始做这个项目”等自然语言表达时，自动按顺序使用 CoBrew 的 `$brainstorm`、`$write-plan`、`$execute-plan` 三个技能。
- 执行时不要求用户记住或手动输入技能名；由助手主动完成梳理、计划和执行。
- 如果任务很小、可以直接完成，则可跳过完整三步流程，直接实现并说明已做的事。
