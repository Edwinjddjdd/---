        三、直升机操纵系统的力学特性
座舱操纵器的运动范围和载荷应当适合驾驶员的生理特性，也就是说，驾驶员在驾驶杆
和脚蹬上感觉到的力学特性，要便于驾驶员精确而顺利地实施操纵动作。操纵系统的力
学特性直接影响到驾驶员的操纵精度和驾驶负荷。

    三、直升机操纵系统的力学特性
传统操纵系统的挑战
旋翼变距力矩直接传到驾驶杆，导致杆力过大
或引起抖振，增加驾驶负荷与风险，难以满足
舒适和精准操作要求。

无回力操纵解决方案
利用液压或电动机构驱动旋翼变距，将旋翼巨
大的操纵力与驾驶杆物理隔离，力矩不再传递
到驾驶杆，从根源解决传统问题。
减轻驾驶负荷：
无需对抗力矩，大幅降低体力消耗。

提升操纵精度：
避免抖振干扰，操控更加平稳精准。

适宜力感反馈：
配备专门机构，提供符合人体工程学的舒适手感。

1、杆力梯度：精准操控的“手感”
  核心定义
  描述操纵杆上的力与位移的关系，决定飞行员的
  “手感”，是实现精准操控的关键物理量。

  理想特性
  •曲线应单调、连续、线性，并基本对称于零位
  •梯度大小适当，便于感知操纵量并轻松回中

  工程应用
  长时间飞行需配备杆力配平系统，俗称调节片,消除
  持续杆力带来的驾驶疲劳。

2、启动力：克服阻力的“第一脚”

  核心定义
  启动力是飞行员开始操纵时，需要克
  服系统摩擦力所施加的初始力。


  成因分析
  • 系统固有的机械摩擦力
  • 防止杆位滑动的设计摩擦
  • 力感弹簧的预压阻力

                                        物理类比：推箱子
  影响与标准                          如同推静止的箱子，需要先施加较大的力克服静摩擦力
  过大导致操纵“发跳”难控制；过小   有摩擦力与启动力的   （启动力）；一旦开始滑动，后续维持运动的力会显著
                                        减小（正常操纵力）。
  则杆位可能自动滑动，需大小适中。    杆力-杆位移特性

3、极限操纵力：人体极限的“边界”

核心定义：在正常飞行状态下，系统允许飞行员施加的最大操纵力。

         悬停 / 低速状态    前飞状态
         纵向：44 N      纵向：133 N
  等级1：   横向：31 N      横向：67 N
         航向：133 N     航向：334 N
         总距：45 N      总距：45 N




   设计目的
 1.防止疲劳：避免因操纵力过大导致飞行员肌肉疲劳，保持长时间飞行的操控稳定性。
 2.应急冗余：确保在紧急情况下，飞行员即使处于生理极限边缘，仍能有效施加足够的力进行操控。

4、空行程：操控响应的“延迟”

   核心定义
 操纵杆已移动，但旋翼等气动面尚未
 做出响应的那段无效行程。




   主要成因
 系统机械间隙、液压管路摩擦阻力、
 自动飞控系统的操纵间断等。

                                            生活中的类比：键盘按键

   负面影响与对策                                就像手指按下键盘，触底前有一段“无效行
                                          程”，直到接触到触发点才会响应。这段
 导致操控响应滞后，降低操纵精度和      操纵系统的力学特性          “空”的距离，就是我们所说的“空行程”。
 飞行品质，设计中需极力减小。
                    驾驶杆的空行程应尽量小(把手处一般
                    应小于0.6mm)，过大则损害操纵性。

Bell 稳定杆




l 贝尔稳定杆由一细长杆及置于两端的质量块构成。
l 当直升机受到扰动时，由于陀螺效应，加大了角速度阻尼，贝尔稳定杆可以自
  动调整周期变距，对旋翼进行二次变距，使直升机稳定。
l 降低了操纵性，对于机动性要求较高的直升机，上述缺点突出。
     五、直升机其他操纵方式
希勒( Hiller)伺服小翼
l 在贝尔杆的质量块处添置短小机
  翼，成为希勒伺服旋翼；
l 与旋翼相比，小翼产生的气动力
  和力矩可以忽略，但小翼的升力
  变化会引起小翼的挥舞运动，借
  助旋翼与小翼之间的机械耦合，
  调节旋翼桨叶的变距；
l 希勒伺服小翼的桨叶洛克数较小，
  与旋翼的挥舞运动在幅值和相位
  上存在差别，从而从幅值和相位
  上影响旋翼的周期变距。
l 来自自动倾斜器的操纵除了对希
  勒旋翼施加周期变距操纵外，同
  时也对旋翼直接施加了周期变距
  操纵，两者的合成产生对旋翼的
  全部周期变距。
