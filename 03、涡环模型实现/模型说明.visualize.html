按学习顺序看

1. vrs_params.m

先认识所有物理参数、无量纲边界和时间常数。

2. vrs_johnson_qs_numeric.m

理解基线诱导速度、VRS 分段曲线和水平速度门控。

3. vrs_johnson_qs.m

从结构体输出观察每个中间结果，便于手算和扫掠。

4. vrs_inflow_rhs.m

理解准定常目标如何通过一阶滞后变成实际诱导速度。

5. test / run / plot

先验证静态曲线，再运行动态场景，最后接六自由度数据。

三条主要使用路线

目的入口主要调用链
看静态公式 test_vrs_static.m params → qs → numeric
看动态进入/改出 run_vrs_testbench.m init → slx → numeric
看已有模型 open_vrs_model.m init → open slx
以后画六自由度结果 plot_vrs_6dof_comparison.m 六自由度数据 → 综合图

当前边界

当前目录只有独立 VRS 测试台，没有六自由度模型。因此 plot_vrs_6dof_comparison.m 目前不参与现有仿真调用链； 它要等六自由度模型产生 t、V_d、T、v_i 后才使用。
