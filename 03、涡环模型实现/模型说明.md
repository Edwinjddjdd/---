# 涡环入流模型说明

## 1. 当前模型做了什么

当前实现是一个独立的涡环平均诱导速度测试台，不包含六自由度机体。它完成：

$$
\left(V_H,V_d,T\right)
\longrightarrow v_{\mathrm{base}}
\longrightarrow v_{i,\mathrm{QS}}
\xrightarrow{\tau\text{ 与积分}} v_i
\longrightarrow q=v_i-V_d.
$$

对应的物理过程是：根据当前速度和拉力计算基线诱导速度，加入 Johnson VRS 修正得到准定常目标，再经过尾流动态滞后得到实际诱导速度，最后与下降造成的相对来流合成总轴向入流。

Simulink 模型为 [`vrs_inflow_testbench.slx`](./vrs_inflow_testbench.slx)。

## 2. 每个方块的物理意义

| 方块 | 物理问题 |
|---|---|
| `Johnson_VRS_QS` | 当前速度和拉力下，如果工况保持不变，诱导速度最终应该是多少？ |
| `Target_Minus_Actual` | 目标诱导速度与当前实际诱导速度还差多少？ |
| `Divide_By_Tau` | 真实尾流应该以多快的速度追赶目标？ |
| `Actual_Induced_Velocity` | 对变化率积分，得到当前实际诱导速度 $v_i$ |
| `Total_Axial_Inflow` | 把诱导速度与下降造成的相对来流合成，得到 $q=v_i-V_d$ |

## 3. 输入与输出

输入：

| 信号 | 含义 | 单位 | 正方向 |
|---|---|---|---|
| `V_H_mps` | 桨盘面内速度 | $\mathrm{m/s}$ | 大小为正 |
| `V_d_mps` | 下降率 | $\mathrm{m/s}$ | 向下为正 |
| `T_N` | 主旋翼拉力 | $\mathrm{N}$ | 正值 |

输出：

| 信号 | 含义 | 单位 |
|---|---|---|
| `v_i_mps` | 实际动态诱导速度 $v_i$ | $\mathrm{m/s}$ |
| `v_i_QS_mps` | 准定常目标诱导速度 $v_{i,\mathrm{QS}}$ | $\mathrm{m/s}$ |
| `vrs_flag` | 当前是否位于 VRS 修正区 | `boolean` |
| `q_mps` | 桨盘总轴向入流 $q=v_i-V_d$ | $\mathrm{m/s}$ |
| `v_base_mps` | 基线诱导速度 $v_{\mathrm{base}}$ | $\mathrm{m/s}$ |
| `dv_vrs_mps` | Johnson VRS 诱导速度修正量 | $\mathrm{m/s}$ |

## 4. 文件作用

| 文件 | 作用 |
|---|---|
| `vrs_params.m` | 集中保存旋翼、环境、Johnson 边界和时间常数参数 |
| `vrs_johnson_qs_numeric.m` | 可供 MATLAB Function 块调用的数值算法 |
| `vrs_johnson_qs.m` | 便于人工阅读和 MATLAB 扫掠的结构体接口 |
| `vrs_inflow_rhs.m` | 一阶动态入流方程 |
| `init_vrs_model.m` | 初始化参数、参考拉力和积分器初值 |
| `test_vrs_static.m` | 静态入流曲线、连接连续性和 N/X 零斜率检查 |
| `build_vrs_model.m` | 重新生成 Simulink 模型 |
| `run_vrs_testbench.m` | 运行进入与水平改出场景并绘图 |
| `open_vrs_model.m` | 初始化并打开顶层模型和 `VRS_Inflow` 子系统 |

## 5. 怎么运行

在 MATLAB 中进入本目录，依次运行：

```matlab
test_vrs_static
build_vrs_model
run_vrs_testbench
```

只想打开模型时运行：

```matlab
open_vrs_model
```

## 6. 当前演示场景

| 时间 | 工况 | 物理意义 |
|---|---|---|
| $0\sim5\ \mathrm{s}$ | $V_H=0$、$V_d=0$ | 悬停 |
| $5\sim15\ \mathrm{s}$ | $V_d$ 增加到 $11\ \mathrm{m/s}$ | 垂直下降进入 VRS |
| $15\sim25\ \mathrm{s}$ | 保持下降率 | 观察 VRS 动态建立 |
| $25\sim35\ \mathrm{s}$ | $V_H$ 增加到 $15\ \mathrm{m/s}$ | 水平速度把尾流带离桨盘 |
| $35\sim40\ \mathrm{s}$ | 保持水平速度 | 检查 VRS 标志退出 |

## 7. 已完成验证

- MATLAB Code Analyzer 对九个实现文件报告 $0$ 条问题。
- 静态下降扫掠输出均为有限实数。
- D、N、X、E 分段连接处连续。
- N、X 点的总轴向入流斜率数值检查接近 $0$。
- $f_{\mathrm{VRS}}=0$ 时 VRS 修正量和标志均关闭。
- Simulink 模型成功编译，共包含 $24$ 个方块。
- 动态场景成功进入 VRS，增加水平速度后最终 `vrs_flag=0`。

## 8. 参数边界

当前使用 Johnson 报告 D6075 算例参数：旋翼半径 $R=5.97\ \mathrm{m}$、转速 $360\ \mathrm{r/min}$、演示质量 $3500\ \mathrm{kg}$。这些数据仅用于跑通模型结构和复现典型趋势，不代表最终目标直升机。

换目标机时首先修改 `vrs_params.m`，不修改 Simulink 连线。Johnson 基线水平速度混合和 D-N-X-E Hermite 分段属于第一版工程实现，最终报告使用前仍需根据原报告曲线和目标旋翼数据继续校核。
